# FitFormulas
Featured calculators:
* Total Daily Energy Expenditure
* Bodyfat Percentage
* Muscular Potential
* Unit Conversion for Lbs-Kgs and Inches-Centimeters

## Built With
* [PyQt5] (https://www.riverbankcomputing.com/software/pyqt/) - GUI
